# 📚 SkillSphere – Online Learning Platform

SkillSphere is a modern full-stack online learning platform built with Next.js, offering secure authentication, protected learning resources, user profile management, and an engaging course experience.

## 🌐 Live Demo
**Live URL:** https://skillsphere-online-learning-platform.vercel.app

---

## ✨ Features

- 🔐 **Authentication System**
  - Email & Password Login
  - Google OAuth Integration

- 🔒 **Protected Routes**
  - Course Details Page
  - User Profile Page

- 🧑‍💼 **User Profile Management**
  - Update Name
  - Update Profile Image

- 📚 **Course Catalog**
  - Browse Courses
  - Search Functionality

- 📄 **Detailed Course Pages**
  - Curriculum Overview
  - Learning Information

- 📌 **Additional Sections**
  - Learning Tips
  - Top Instructors

- 🏆 **Popular Courses**
  - Top 3 Highest Rated Courses

- 🔔 **Interactive Toast Notifications**

- 📱 **Responsive Design**
  - Mobile
  - Tablet
  - Desktop
